# ImageCrop

Simple proxy application to resize images on the fly.

## Use

Enter your query in following format:
{domain}/resize?width={width}&height={height}&url={url}

## Features

- Redis image caching.
- Remote image filesize validation.
- Image bomb protection.
- Filesize check.
